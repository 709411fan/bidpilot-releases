# BidPilot 安装指南 · Windows

> ⚠️ **本文档为占位版 · 完整安装指南将在 Sprint I-A · B7 阶段正式发布。**

---

## 预期系统要求（待 B7 最终确认）

- Windows 10 / 11 · 64 位
- WPS Office 2019+ · 专业版 / 校园版 / 个人免费版均可
- 网络连接（首次启动需要）
- 建议预留 2 GB 磁盘空间

---

## 预期安装流程（草案 · 待 B7 最终确认）

1. 访问 [intelgenai.com](https://intelgenai.com/products/bidpilot) 或直接到本仓库 [Releases](https://github.com/709411fan/bidpilot-releases/releases)
2. 下载最新的 `BidPilotSetup-vX.Y.Z-alpha.N.exe`
3. 双击 installer · 按向导完成安装（默认安装到 `%LOCALAPPDATA%\BidPilot\`）
4. 启动 WPS（Writer 打开任意文档）· 看到 "工书" tab 说明成功
5. 点击任意按钮 · 右侧弹出插件面板 · 等待首次启动

详细步骤、截图、常见问题、故障排查 → **待 Sprint I-A · B7 阶段补全**。
